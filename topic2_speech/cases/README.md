# Case Audio

This folder contains the four case clips used by the report and the MATLAB demos.

- `case_g_get.wav` -> `/g/`, source word `get`
- `case_b_boy.wav` -> `/b/`, source word `boy`
- `case_d_did.wav` -> `/d/`, source word `did`
- `case_z_zero.wav` -> `/z/`, source word `zero`

`boy` is used for `/b/` because the current MSWC subset does not contain `book`.

Files:

- `case_manifest.csv`: source mapping for the four clips

MATLAB entry points:

- `../src/matlab_case_demo.m`
- `../src/matlab_course_template_detector.m`
- `../src/matlab_course_spectrum_demo.m`
